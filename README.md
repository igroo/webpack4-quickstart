# Webpack4 Quick Start

Web Front 개발을 하기 위하여 Webpack4 구성 하였다. 

## Getting Started

webpack-dev-server 를 사용한 환경

```
npm run start
```

Development Build
```
npm run dev
```

Production Build
```
npm run build
```
